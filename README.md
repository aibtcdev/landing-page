## Development

Tech stack:

- React
- Vite
- Chakra UI

To run this locally:

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open [http://localhost:5173](http://localhost:5173) with your browser

Deployments are handled by Cloudflare Pages.
